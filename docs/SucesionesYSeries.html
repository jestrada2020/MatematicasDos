Capítulo 6 Sucesiones y Series

6.1 Concepto de sucesión

Definición 6.1 Una sucesión es una aplicación (ó función) entre el conjunto de los números naturales y los números reales.

\[ N \ \ \ \ \longrightarrow \ \ R\\ \ \ \ \ \ \ \ \ n \ \ \ \longmapsto \ a_{n}=f(n) \]

Diremos que la sucesión \(\{a_{n}\}\) converge si el limite del término n-ésimo \(a_{n}\) existe
\[ \lim_{n \to \infty}{a_{n}}=L \]

De no existir el límite diremos que la sucesión \(\{a_{n}\}\) diverge.



Definición 6.2 Una sucesión \(\{a_{n}\}\) converge a un límite \(L\)

si para todo \(\epsilon >0\) hay un correspondiente \(N>0\) tal que

\[ \text{Si} \ \ n>N \ \ \ \text{entonces} \ \ \ |a_{n}-L|< \epsilon \]



Teorema 6.1 Si \(\lim_{x \to \infty}{f(x)}=L\) y \(f(n)=a_{n}\) cuando \(n\) es un entero, entonces \(\lim_{n \to \infty}{a_{n}}=L\)

Definición 6.3 Si el \(\lim_{n \to \infty}{a_{n}}=\infty\) significa que para todo número positivo \(M\) existe un entero \(N\) tal que:

\[ \text{Si} \ \ n>N \ \ \ \text{entonces} \ \ \ a_{n} > M \]

Es decir la sucesión \(\{a_{n}\}\) es divergente pero de una menera especial. Se dice que \(\{a_{n}\}\) diverge a \(\infty\)

6.2 Leyes de los límites para las sucesiones

Teorema 6.2 Si \(\{a_{n}\}\) y \(\{b_{n}\}\) son sucesiones convergentes y \(c\) es una constante, entonces

  1. \(\lim_{n \to \infty}{(a_{n}+b_{n})}=\lim_{n \to \infty}{a_{n}}+\lim_{n \to \infty}{b_{n}}\)

  2. \(\lim_{n \to \infty}{(a_{n}-b_{n})}=\lim_{n \to \infty}{a_{n}}-\lim_{n \to \infty}{b_{n}}\)

  3. \(\lim_{n \to \infty}{ca_{n}}=c\lim_{n \to \infty}{a_{n}}\)

  4. \(\lim_{n \to \infty}{c}=c\)

  5. \(\lim_{n \to \infty}{(a_{n}.b_{n})}=\lim_{n \to \infty}{a_{n}}.\lim_{n \to \infty}{b_{n}}\)

  6. \(\lim_{n \to \infty}{\left(\dfrac{a_{n}}{b_{n}}\right)}=\dfrac{\lim_{n \to \infty}{a_{n}}}{\lim_{n \to \infty}{b_{n}}}\) si \(\lim_{n \to \infty}{b_{n}}\) es diferente de cero.

  7. \(\lim_{n \to \infty}{\left({a_{n}}\right)^q}={\left(\lim_{n \to \infty}{a_{n}}\right)^q}\) si \(q>0\) y \(a_{n}>0\)

6.3 El teorema de la compresión para sucesiones

Teorema 6.3 Si \(a_{n}\leq b_{n} \leq c_{n}\) para todo \(n>n_{0}\) y \(\lim_{n \to \infty}{a_{n}}=\lim_{n \to \infty}{c_{n}}=L\), entonces \(\lim_{n \to \infty}{b_{n}}=L\)

Teorema 6.4 Si \(\lim_{n \to \infty}{|a_{n}|}=0\), entonces \(\lim_{n \to \infty}{a_{n}}=0\)

Teorema 6.5 Si \(\lim_{n \to \infty}{a_{n}}=L\) y la función \(f\) es continua en \(L\), entonces \(\lim_{n \to \infty}{f(a_{n})}=f(L)\)

Definición 6.4 Una sucesión \(\{a_{n}\}\) es monótona si sus términos no son decrecientes

\[ a_{1} \leq a_{2} \leq a_{3} \leq a_{1} \dotsb \leq a_{n-1}\leq a_{n} \dotsb \]

o si sus términos no son crecientes

\[ a_{1} \geq a_{2} \geq a_{3} \geq a_{1} \dotsb \geq a_{n-1}\geq a_{n} \dotsb \]

  1. Ejemplo sucesión que si es monotonas

\[ a_{n}=\dfrac{2n}{1+n} \]

  1. Ejemplo sucesión que no es monotona

\[ a_{n}=(-1)^{n}+3 \]

\[ a_{n}=\dfrac{n^2}{2^n-1} \]

Nota: Otra manera de argumentar que la sucesión es monótona es usando la derivada y verificando que su signo es positivo para todo \(x\geq0\) ó que el signo de la derivada es negativo para todo \(x\geq0\).

Definición 6.5 Una sucesión \(\{a_{n}\}\) es acotada superiormente si existe un número real \(M\) tal que \(a_{n}\leq M\) para todo \(n\). El número real \(M\) es llamado una cota superor de la sucesión.

Una sucesión \(\{a_{n}\}\) es acotada inferiormente si existe un número real \(N\) tal que \(N \leq a_{n}\) para todo \(n\). El número real \(N\) es llamado una cota inferior de la sucesión.

Una sucesión \(\{a_{n}\}\) es acotada si lo esta superior e inferiormente.

Teorema 6.6 Si una sucesión \(\{a_{n}\}\) es monótona y acotada, entonces es convergente.
